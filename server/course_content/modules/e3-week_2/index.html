E3 Week 12: Instrumentation Strategy, Signal Conditioning & Beyond-Provided Hardware

With experimental design principles in place, Week 12 examines how sensor and instrumentation choices shape the credibility of the entire validation effort. Students evaluate sensor tradeoffs — dynamic range, resolution vs. accuracy, mounting effects, environmental contamination, resource constraints — and learn to select instrumentation matched to the accuracy requirements of their intended use. Learning Goals

  • Select sensors for an original aerospace experiment with written rationale connecting sensor specifications to accuracy requirements and intended use — CLO 1
  • Specify DAQ configuration and signal conditioning strategy for an original experiment, including sampling rate justified by Nyquist argument and anti-aliasing filter cutoff justified relative to signal bandwidth — CLO 1, CLO 4
  • Use MATLAB-based uncertainty analysis to predict the measurement performance of a proposed sensor and DAQ configuration before any hardware is selected — CLO 2
  • Investigate at least one sensor or measurement platform not provided in the course and evaluate whether it offers meaningful advantages for a specific measurement problem — CLO 6
  • Cite engineering standards and manufacturer specifications as authoritative sources for sensor performance claims in a formal instrumentation justification report — CLO 1, CLO 6

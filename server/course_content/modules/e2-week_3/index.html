E2 Week 3 (Course Week 8)

E2 Week 8: Model Adequacy, Discrepancy & Calibration Revisited

Phase I taught students to ask whether model and experiment agree. Phase II asks a harder question: when they disagree, what does that tell you about the model? Week 8 introduces model-form error, sensitivity to boundary condition assumptions, and the distinction between discrepancy as failure and discrepancy as diagnostic information. Learning Goals

  • Interpret model-experiment discrepancy as diagnostic information about model-form limitations rather than experimental failure — CLO 3
  • Distinguish between parametric uncertainty (imprecise knowledge of model parameters), model-form error (structural inadequacy of the model), and measurement error — CLO 2, CLO 3
  • Inspect residuals structurally — identifying systematic trends vs. random scatter — and articulate what each pattern implies about model adequacy — CLO 3
  • Perform a calibration status review with reference to instrument documentation and distinguish calibration activities from validation activities — CLO 1, CLO 3
  • Use experimental results to propose specific improvements to a predictive model — identifying what model parameters or assumptions the data reveals are inaccurate or incomplete — CLO 3
  • Begin the Tier 2 group report analysis sections using the released Canned Experiment 1 data — CLO 5

Learning Goals

  • Read a physical model's equations to identify what must be measured, how precisely, and under what conditions — connecting model structure to experimental design requirements — CLO 1, CLO 3
  • Rank the contributions of multiple error sources in a measurement chain and articulate where improving one input would most reduce total uncertainty in the derived output — CLO 2
  • Execute a complete Phase I experimental pipeline — hypothesis, sensor selection, installation, MATLAB-based data collection, uncertainty propagation, and quantitative model comparison — as a single traceable workflow — CLO 1, CLO 2, CLO 3
  • Write explicit sensor selection rationale connected to accuracy requirements and intended use — CLO 1
  • Perform a structured calibration procedure and propagate calibration uncertainty into the final experimental outcome — CLO 1
  • Produce a complete Tier 1 Phase 1 Individual Report with all required sections, AMVF annotation, and properly captioned MATLAB figures — CLO 5

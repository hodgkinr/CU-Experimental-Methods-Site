E1 Week 3: Quantitative Comparison, Validation Basics & Multiple Sensors

With uncertainty quantified, Week 3 addresses the question left open by Week 2: how do you use that uncertainty to argue whether your result agrees with a model? Visual comparison of curves is insufficient — agreement must be argued numerically using a defined validation metric. Learning Goals

  • Apply a named validation metric to compare a measured outcome to a model prediction within defined uncertainty bounds — CLO 2, CLO 3
  • Explain the distinction between calibration and validation and articulate why conflating them undermines experimental credibility — CLO 3
  • Evaluate agreement between two measurements of the same quantity using different sensors or methods — CLO 1, CLO 2
  • Identify at least one model-form limitation from residual inspection and articulate its implication — CLO 3
  • Write a comparative analysis memo with a named validation metric, numerical conclusion, and uncertainty-bounded argument — CLO 5
  • Evaluate a provided lab report — identifying what is correct, what is incorrect, what is missing, and how to improve it — demonstrating that critique quality is a primary course competency — CLO 5

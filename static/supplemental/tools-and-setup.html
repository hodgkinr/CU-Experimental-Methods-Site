MATLAB Mobile and Vision Tools

MATLAB Mobile gives students a fast path to phone-based sensor data collection. In Tier 1 work, it is a practical way to stream and log accelerometer or orientation data without dedicated hardware.

Computer vision tools provide a second path: using video as a sensor. That matters in this course because one recurring design principle is measuring the same quantity with more than one sensing method.

The important course idea is not "use the fanciest sensor." It is "choose a sensing approach you can justify."

Overleaf and LaTeX

Some course deliverables are submitted as LaTeX source plus a compiled PDF. For students who are new to LaTeX, the goal is not mastery of the whole ecosystem. The goal is a working command of sections, figures, equations, and clean technical formatting.

The minimum set of tasks students usually need is:

  • create sections and subsections
  • insert and caption figures
  • write equations
  • fix basic syntax errors when the document does not compile

Portfolio Mindset

Optional portfolio-building artifacts can be useful when they genuinely show engineering reasoning, not just polished packaging.

That means a repository history, a short presentation, or a clear technical summary can all be valuable if they show how the work was defended, revised, and communicated.

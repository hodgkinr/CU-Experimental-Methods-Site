# ASEN 3501 Status Tracker

This tracker lives in `static/admin/` because the current course content in this repo is organized under `static/` rather than `claude_V1/`.

## Legend

`?` means the status is an informed guess based on the repo contents.

For readings and lectures:

- `D` = drafted
- `R` = reviewed
- `L` = lightboard recorded
- `E` = edited
- `C` = Canvas / ready for production

For assignments:

- `D` = drafted
- `R` = reviewed
- `L` = LaTeX / Overleaf ready
- `C` = Canvas / ready for production
- `A` = AI grader ready

Blank status means no concrete source artifact was found yet and the row is acting as a planning placeholder.

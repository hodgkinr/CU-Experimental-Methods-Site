Procedure — Aeroelasticity Lab (Compiled First Pass)

This document consolidates the experimental procedures for all three phases of the aeroelasticity lab sequence. A student or lab assistant who has never seen the source documents should be able to run these experiments using only this file, together with the sensor and DAQ details in instrumentation/sensor_specs_summary_agent.md.

The lab has two parallel experimental tracks that eventually converge:

  • Aerodynamics track: Measures airspeed and airfoil pressure distributions using the desktop and PILOT wind tunnels.
  • Structures track: Designs and tests a whiffle tree apparatus, then measures wing deflection.

Part 1 — Desktop Wind Tunnel Data Collection (Aerodynamics Track, Phase 1)

Source: Milestone 1, Appendix B

Purpose: Measure differential pressure and calculate airspeed at various positions in the desktop wind tunnel; collect data in three configurations to understand Pitot-Static and static-only measurements.

Setup

  1. Connect the Arduino to the computer using the USB-A cable.
  2. In MATLAB, type arduinosetup in the command window.
  • Select "Uno" for the board type.
  • Record the port number (appearance varies by operating system).
  • Deselect all server libraries and click "Program."
  • On the following page, click "Test arduino connection."
  • If the Arduino is working correctly, exit this window.
  1. From the Honeywell ABP2 datasheet (see instrumentation/sensor_specs_summary_agent.md), determine the scale factor k (units: Pa/V). [INSTRUCTOR TO PROVIDE: scale factor k — this value is set by the instructional team, not read directly from the datasheet.]
  2. In MATLAB, type WindTunnelGUI and click "Run Section" to start the GUI.
  3. Enter the Serial Port Number and Scale Factor into the GUI.
  4. Click "Start Data Collection." Wait 5–10 seconds, then click "Zero Pressure Transducer."
  5. After the GUI reports the offset, click "Stop & Save Data," then "Reset Data," and restart data collection to verify the zero offset.

Data Collection Procedure

  1. With lab assistant approval, activate the fan and turn the voltage control knob to its maximum setting.
  2. Collect data in three configurations in sequence:
  • Configuration 1: Connect the total pressure port (p₀) and static pressure port (p_s) to the transducer. Move the probe along the length of the inlet to the back of the test section, observing how differential pressure changes. Select the appropriate configuration in the GUI drop-down. When complete, click "Stop & Save Data."
  • Configuration 2: Disconnect the static pressure port only. Reset the data. Select the appropriate configuration in the GUI drop-down. Repeat the same probe traverse. Save data.
  • Configuration 3: Connect only the static pressure port. Select the appropriate configuration in the GUI drop-down. Repeat the traverse. Save data.
  1. After collecting sufficient data, reduce the fan voltage control to zero.
  2. Load data by double-clicking on any of the three files in the current folder or typing load [filename] in MATLAB.

[SETUP_PHOTO NEEDED: Desktop wind tunnel with Pitot-Static probe inserted, Arduino cable visible, and MATLAB GUI displayed on screen]

Pitfalls — Instructor Notes

(From "Questions to Consider" in M1 §5.2 and §5.3)

  • Students often confuse which port measures total vs. static pressure. Reinforce the cross-sectional diagram of the Pitot-Static probe.
  • The area ratio between the settling chamber and the test section matters significantly for the Venturi Tube configuration; students should understand why airspeed differs between inlet and test section.
  • Students may expect the differential pressure to be the same at all probe positions — it is not. The profile across the inlet and test section is part of the learning objective.
  • The choice between the Pitot-Static and Venturi configurations involves a trade-off: the Pitot probe causes blockage; the Venturi avoids probe intrusion but requires area ratio knowledge.

Part 2 — Clark Y-14 Infinite Wing Pressure Survey (Aerodynamics Track, Phase 2)

Source: Milestone 2, Appendix C

Purpose: Measure the chord-wise static pressure distribution on a Clark Y-14 airfoil at multiple angles of attack and airspeeds using the PILOT low-speed wind tunnel.

Note: All measurements are acquired by the LabVIEW control program at 500 Hz for a period of 0.04 s (20 measurements per collection point). Data is saved in collimated ASCII format.

Setup

  1. Visually inspect the PILOT tunnel and verify its configuration.
  2. Locate the pressure tubes for the pitot-static tube. Confirm they are connected to the airspeed pressure transducer so the wind tunnel control computer can monitor tunnel speed.
  3. Inspect the airfoil model: verify that only the correct set of spanwise pressure ports is open and all others are covered with tape.
  4. Connect all pressure tubes from the airfoil model to the Scanivalve pressure scanner. Confirm the measurement configuration of each port.
  5. Using the angle markings on the airfoil base plate, set the airfoil to the first assigned angle of attack. Tighten all retaining screws securely.
  6. Close and lock the wind tunnel door.

[SETUP_PHOTO NEEDED: PILOT wind tunnel test section with Clark Y-14 airfoil mounted, angle markings visible, and Scanivalve connections visible]

Procedure

  1. Open the LabVIEW control program: WindTunnel - Shortcut.vi (located in AES Courses Folder: Courses/AES/Software/VIs/ASEN2002). Click the white arrow in the top left to start the VI.
  2. Set your data file path using the assigned naming convention. Do not overwrite another group's data file.
  3. Zero the airspeed and pressure transducers using the green button on the VI front panel (bottom left).
  4. Open the angle of attack window; enter the current angle of attack.
  5. Switch the control mode toggle to "airspeed control."
  6. Enter the test velocity into the LabVIEW VI to bring the fan up to speed.
  7. Confirm measurement parameters: Sampling Rate = 500 Hz, Samples per Set = 20.
  8. Once the tunnel reaches equilibrium (typically 30–60 s), collect measurements by clicking "Send Samples to File."
  9. Return airspeed to 0 m/s. Do NOT stop the VI.
  10. Once the tunnel has settled, open the tunnel door. Rotate the airfoil to the next angle of attack. Secure the model. Close the door.
  11. Repeat steps 12–16 for each assigned velocity and angle of attack, progressing through angles in the assigned order. Always return to zero airspeed before changing angle of attack.
  12. After all data collection is complete, return the tunnel to zero, wait for the tunnel to fully stop, then stop the VI to save and close the data file.
  13. Immediately upload data to the class folder.

Pitfalls — Instructor Notes

(From "Questions to Consider" in M2 §5.2–5.4)

  • Students should notice that pressure port spacing is non-uniform — denser near the leading edge where pressure gradients are steepest.
  • The trailing edge pressure cannot be read directly; it must be extrapolated (linear regression from ports 8→9 and 10→11, then averaged). Students often try to average the last two ports directly, which is incorrect.
  • The Cp scale is conventionally inverted (negative up); students must flip their plots.
  • Lift at zero angle of attack is non-zero for a cambered airfoil — this surprises many students.

Part 4 — Whiffle Tree Assembly and Bench Test (Structures Track)

Source: Milestone 3, §6.1 and Appendix A

Purpose: Experimentally verify the analytical deflection model by applying a simulated distributed load to a spar beam via a 4-point whiffle tree, and measuring the resulting tip deflection.

Pre-requisite: Whiffle tree geometry (dimensions a–f) must be computed from discretizeLoad.m and whiffleTree4.m before assembly. External mass must be calculated.

Materials

  • Two aluminum spar beams, 1/8" × 1/2" × 12"
  • Steel rod whiffles (3 bars)
  • Paper clips (4, for attaching whiffle tree to spar)
  • String or fishing wire
  • Duct tape
  • C-clamp
  • Chair
  • Masses (calculated from whiffle tree design)
  • Ruler

Setup

The spar beams overhang a table and are secured with a C-clamp. The whiffle tree is attached to one spar using paper clips and secured with duct tape. The second spar is unloaded and used as a reference. External masses are tied together as a single external mass. A ruler is positioned vertically to measure the deflection of the loaded spar relative to the unloaded spar.

[SETUP_PHOTO NEEDED: Whiffle tree assembly attached to spar beam, C-clamp visible, ruler positioned at tip, reference spar alongside]

Procedure

  1. Acquire all materials.
  2. Tape paper clips to the whiffles at the computed positions.
  3. Tape the completed whiffle tree to the spar beam.
  4. Clamp the spar beam to the table using the C-clamp, ensuring the spar overhangs by exactly 10 inches.
  5. Position the ruler vertically next to the spar tip.
  6. Record the initial position of the loaded spar tip relative to the unloaded spar.
  7. Tie the required external mass together using fishing wire or string.
  8. Place a chair under the spar and whiffle tree.
  9. Secure the external mass to the bottom whiffle, resting the mass on the chair.
  10. Remove the chair, allowing the external mass to hang freely.
  11. Measure the spar tip position relative to the unloaded spar. Calculate deflection from the difference.

Also test: Deflection under a single point load acting at the centroid of the load distribution, for comparison against the distributed load result.

Pitfalls — Instructor Notes

  • Paper clip attachment points must exactly match the computed x-positions (x₁–x₄); small deviations change the load distribution significantly.
  • The spar must overhang exactly 10 inches (matching L used in calculations); use a ruler to verify before clamping.
  • Masses should be assembled and weighed before the test to confirm they match the computed Feq.

Part 5 — Wind Tunnel Deflection Test (Structures Track)

Source: Milestone 3, §6.2 and Appendix B

Purpose: Measure the actual aerodynamic deflection of the Clark Y-14 finite wing in the PILOT wind tunnel at a specific angle of attack and airspeed, for comparison to the analytical model.

Test condition: Angle of attack = 10°; test velocity = 30 m/s.

Test Specimen

Clark Y-14 deflection specimen: PLA plastic ribs on a 1/8" × 1/2" × 12" aluminum spar, wrapped in clear packing tape. Span = 10". The wing is secured at the base by the spar only. The wing profile begins within the wind tunnel boundary layer, so the wing is analyzed as a half-finite airfoil.

Setup

The wing is mounted horizontally in the PILOT tunnel, with the positive z-direction oriented upright. A ruler is mounted on the front door of the wind tunnel for measuring vertical deflection.

[SETUP_PHOTO NEEDED: Clark Y-14 deflection specimen mounted in PILOT tunnel test section, ruler visible on tunnel door, wing horizontal with spar root visible]

Procedure

  1. Visually inspect the tunnel; verify that there are no major geometric defects in the airfoil.
  2. Using the angle markings on the airfoil base plate, set the airfoil to 10°. Tighten all retaining screws securely.
  3. Close and lock the wind tunnel door.
  4. Attach masking tape perpendicular to the chord line, positioned at the quarter chord. Use a machinist's protractor to align the tape.
  5. Mark the initial vertical position of the lowest point of the spar beam on the spar beam (no-load reference).
  6. Open the LabVIEW control program (WindTunnel - Shortcut.vi) as described in Part 2.
  7. Set the data file path (do not overwrite other files).
  8. Zero the airspeed and pressure transducers using the green button.
  9. Enter 10° in the angle of attack window.
  10. Switch to "airspeed control" mode.
  11. Enter 30 m/s as the test velocity to bring the tunnel up to speed.
  12. Once the tunnel has reached equilibrium (30–60 s), record the new vertical position of the highest point of the spar beam (loaded condition).
  13. Return the wind tunnel to 0 m/s. Stop the VI.
  14. Calculate deflection from the difference in loaded and unloaded spar tip positions.

Pitfalls — Instructor Notes

  • The ruler position must be carefully recorded relative to a fixed reference before and after tunnel operation; the tunnel vibrates slightly during operation.
  • The spar root must be the only structural connection to the tunnel — ribs and skin transfer load to the spar, not to the tunnel walls.
  • Confirm that the masking tape reference mark is on the spar, not the skin, since skin flexes independently.

Model / Derivation

Theoretical Framework — Aeroelasticity Lab (Compiled First Pass)

This document extracts the governing equations, derivations, and stated assumptions from the three Milestone source documents. It is organized by topic rather than by source document page order. All equations are preserved as stated in the source; symbols are defined immediately following each equation.

1. Airspeed Measurement from Differential Pressure

Source: Milestone 1, §5.3 and Appendix A

1.1 Bernoulli's Equation for Incompressible Flow

The starting point is Bernoulli's equation applied along a streamline in the wind tunnel:

$$p_1 + \frac{1}{2}\rho V_1^2 = p_2 + \frac{1}{2}\rho V_2^2 = p_3 + \frac{1}{2}\rho V_3^2$$

Assumption: Incompressible flow (density ρ is constant throughout the tunnel).

1.2 Ideal Gas Law for Air Density

Air density cannot be measured directly. Using the equation of state for a perfect gas:

$$\rho = \frac{p}{RT} = \frac{p_\text{atm}}{R T_\text{atm}}$$

SymbolDescriptionUnits
ρAir densitykg/m³
p_atmAtmospheric pressurePa
T_atmAtmospheric temperatureK
RSpecific gas constant for air (287 J/(kg·K))J/(kg·K)

Assumption: Atmospheric conditions (measured outside the tunnel under stagnant conditions) represent the density throughout the tunnel.

1.3 Pitot-Static Probe Configuration (Equation 1)

For a Pitot-Static probe, the stagnation condition (V₀ → 0 at the total pressure port) gives:

$$p_0 - p_s = \Delta p = q = \frac{1}{2}\rho V_\infty^2 = \frac{1}{2}\frac{p_\text{atm}}{RT_\text{atm}}V_\infty^2$$

Rearranging to solve for airspeed:

$$V_\infty = \sqrt{\frac{2(p_0 - p_s)}{\rho}} = \sqrt{\frac{2\Delta p}{\rho}} = \sqrt{\frac{2\Delta p \, R T_\text{atm}}{p_\text{atm}}} \tag{1}$$

SymbolDescriptionUnits
V_∞Freestream airspeed (test section)m/s
p₀Total (stagnation) pressurePa
p_sStatic pressurePa
ΔpDifferential pressure (p₀ − p_s)Pa
qDynamic pressurePa

Assumptions:

  • Incompressible flow
  • Negligible losses between total and static measurement locations

1.4 Venturi Tube Configuration (Settling Chamber–Test Section)

The second standard configuration compares pressures at two locations. Using mass conservation and Bernoulli:

$$V_\infty = \sqrt{\frac{2(p_{s,1} - p_{s,2})}{\rho\left(1 - \left(\frac{A_2}{A_1}\right)^2\right)}} = \sqrt{\frac{2\Delta p \, R T_\text{atm}}{p_\text{atm}\left(1 - \left(\frac{A_2}{A_1}\right)^2\right)}}$$

SymbolDescription
p_{s,1}Static pressure in settling chamber
p_{s,2}Static pressure in test section (= p_{s,∞})
A₁Cross-sectional area of settling chamber
A₂Cross-sectional area of test section

Note: For the desktop wind tunnel: A₁ is the inlet area (4.3" diameter), A₂ is the test section area (1.125" diameter).

2. Airspeed Uncertainty Propagation

Source: Milestone 1, §5.4

Applying the general uncertainty propagation formula from Taylor (§3.11):

$$\delta q = \sqrt{\left(\frac{\partial q}{\partial x}\delta x\right)^2 + \cdots + \left(\frac{\partial q}{\partial z}\delta z\right)^2}$$

For airspeed V_∞ as a function of Δp, p_atm, and T_atm:

$$\delta V = \sqrt{\left(\frac{\partial V}{\partial \Delta p}\,\delta\Delta p\right)^2 + \left(\frac{\partial V}{\partial p_\text{atm}}\,\delta p_\text{atm}\right)^2 + \left(\frac{\partial V}{\partial T_\text{atm}}\,\delta T_\text{atm}\right)^2} \tag{2}$$

SymbolDescription
δVTotal uncertainty in computed airspeed
δΔpUncertainty in differential pressure measurement (from pressure transducer total error band; reference off full collection range)
δp_atmUncertainty in atmospheric pressure measurement
δT_atmUncertainty in atmospheric temperature measurement

Key point: The partial derivatives of V with respect to each variable must be computed analytically from Equation 1 before this formula can be evaluated. [STUDENT TASK — students derive the partial derivatives and evaluate δV.]

3. Pressure Coefficient and Aerodynamic Force Coefficients

Source: Milestone 2, §5.3–5.4, Appendix B, Appendix D

3.1 Pressure Coefficient C_p (Equations 1 and 13)

$$C_p = \frac{\Delta p}{q_\infty} = \frac{p - p_\infty}{\frac{1}{2}\rho_\infty V_\infty^2} \tag{1/13}$$

Alternative form using Bernoulli:

$$C_p = 1 - \frac{V^2}{V_\infty^2}$$

SymbolDescription
C_pPressure coefficient (dimensionless)
pLocal static pressure at a surface pointPa
p_∞Freestream static pressurePa
q_∞Freestream dynamic pressure = ½ρ_∞V_∞²Pa
VLocal flow speed at the surfacem/s

Note on sign convention: C_p plots are conventionally displayed with the C_p axis inverted (negative values plotted upward).

Trailing-edge estimation: There is no physical port at the trailing edge. Estimate the trailing edge pressure using linear extrapolation from adjacent ports on the upper surface (ports 8→9) and lower surface (ports 10→11), then average the two extrapolated values.

3.2 Normal and Axial Force Coefficients — Elemental Form (Equations 20–25, 27–28)

For an element spanning pressure taps i and i+1, with element geometry Δxᵢ = xᵢ₊₁ − xᵢ and Δyᵢ = yᵢ₊₁ − yᵢ:

Normal force element (in dimensional form):

$$n_i = -\frac{1}{2}(p_i + p_{i+1})\Delta x_i \tag{21}$$

Axial force element (in dimensional form):

$$a_i = \frac{1}{2}(p_i + p_{i+1})\Delta y_i \tag{23}$$

Summing over all elements and non-dimensionalizing by ½ρ_∞V_∞²c:

$$C_n = -\sum_{i=1}^{n}\frac{1}{2}(C_{p_i} + C_{p_{i+1}})\frac{\Delta x_i}{c} \tag{27}$$

$$C_a = \sum_{i=1}^{n}\frac{1}{2}(C_{p_i} + C_{p_{i+1}})\frac{\Delta y_i}{c} \tag{28}$$

These are trapezoidal rule approximations to the exact contour integrals:

$$C_n = -\oint C_p \frac{dx}{c}, \qquad C_a = \oint C_p \frac{dy}{c}$$

SymbolDescription
C_nSectional normal force coefficient
C_aSectional axial force coefficient
cAirfoil chord length
(xᵢ, yᵢ)Chord-wise and thickness-wise position of pressure tap i

Coordinate system: x-axis is chordwise (leading edge to trailing edge); y-axis is in thickness direction; z-axis is spanwise. Origin is at the quarter-chord.

3.3 Lift and Drag Coefficients (Equations 4–5, 31–32)

Transform from airfoil frame to freestream frame:

$$C_l = C_n\cos\alpha - C_a\sin\alpha \tag{4/31}$$

$$C_d = C_n\sin\alpha + C_a\cos\alpha \tag{5/32}$$

SymbolDescription
C_lSectional lift coefficient
C_dSectional drag coefficient (pressure drag only)
αAngle of attackrad

Note: These are sectional (per unit span) coefficients for an infinite rectangular wing.

3.4 Lift and Drag for Finite Wing

For a finite rectangular wing with span b and chord c, the reference area S = c·b, and the wing-level coefficients are:

$$C_L = \frac{L}{\frac{1}{2}\rho_\infty V_\infty^2 S}, \qquad C_D = \frac{D}{\frac{1}{2}\rho_\infty V_\infty^2 S}$$

For a rectangular wing with uniform spanwise distribution, C_L = C_l and C_D = C_d.

4. Load Discretization and Whiffle Tree Design

Source: Milestone 1, §6.1–6.3

4.1 Wing Spar as a Cantilevered Beam

The wing spar is modeled as a flexible cantilevered beam:

  • Fixed (welded) connection at the root: x = 0
  • Free tip at x = L = 10 in
  • Beam length L = 10 in (to fit within 12" × 12" PILOT test section)
  • Gravity ignored (weight of beam assumed negligible compared to lift load)

Three load distributions are analyzed:

  • Rectangular: uniform load, magnitude ω₀ (lbf/in)
  • Triangular: linearly decreasing, ω(x) = ω₀(1 − x/L)
  • Trapezoidal: ω(x) = ω₀ + (ω₁ − ω₀)(x/L) where ω₀ and ω₁ are root and tip intensities

Note: Values of ω₀ and ω₁ are determined from the aerodynamic analysis (Milestone 3). Initial whiffle tree designs are carried out symbolically.

4.2 Load Discretization using Trapezoidal Rule

The discretizeLoad MATLAB function divides the span into N panels, applies the trapezoidal rule to each panel, and returns:

  • N×1 vector of resultant panel forces (magnitudes)
  • N×1 vector of centroid x-locations of each panel

Each panel force equals the area under the load curve over that panel width; the centroid is the weighted centroid of that trapezoidal strip.

4.3 Whiffle Tree Equivalent Force and Centroid (Equation 3)

The equivalent total force and its location from the root:

$$x_{F_{eq}} = \frac{\sum F_{Ri}\, x_i}{\sum F_{Ri}} \tag{3}$$

SymbolDescription
F_{Ri}Resultant force of the ith panellbf
xᵢx-location of the ith panel centroidin
x_{Feq}x-location of the equivalent total forcein
F_{eq}Magnitude of equivalent total force = ΣF_Rilbf

4.4 Whiffle Tree Geometry

The whiffle tree has 4 connecting loads to the spar beam and 3 bars. Geometric variables a–f describe the bar lengths and spacings. These are solved by writing moment equilibrium equations for each bar, putting them into matrix form, and solving with MATLAB.

5.2 Second Moment of Area (Equation 12)

For a thin rectangular cross-section, bending about the y-axis (relevant to out-of-plane deflection):

$$I_y = \frac{wt^3}{12} \tag{12}$$

Substituting: I_y = (0.5)(0.12)³/12 = 7.2 × 10⁻⁵ in⁴

5.3 Beam Curvature and Bending Stress (Equations 9–11)

The second derivative of deflection ν(x) equals the ratio of bending moment to bending rigidity:

$$\nu''(x) = \frac{d^2\nu(x)}{dx^2} = \frac{M(x)}{EI} \tag{9}$$

Internal bending stress:

$$\sigma(x,z) = -\frac{M(x)z}{I} \tag{10}$$

Bending strain (Hooke's law):

$$\varepsilon(x,z) = \frac{\sigma(x,z)}{E} \tag{11}$$

5.4 Internal Bending Moment — Distributed Load

Two methods available for finding M(x):

  1. Method of Cuts: Make a cut at arbitrary spanwise position x; solve for M(x) from equilibrium of the cut segment.
  2. Integration Method: V(x) = −∫ω(x)dx; M(x) = −∫V(x)dx, using boundary conditions at the fixed end (x=0) and free end (x=L).

[STUDENT TASK — students derive M(x) for rectangular, triangular, and trapezoidal loads in ASEN 3501.]

5.5 Point Load Deflection (Equation 6)

Source: Appendix E (full derivation reproduced in Milestone 2 source)

The maximum deflection of a cantilevered beam due to a point load F_eq acting at location x̃ (centroid of the distributed load):

$$\nu_{\max,\text{point load}} = \frac{F_{eq}\tilde{x}^2}{6EI}(3L - \tilde{x}) \tag{6}$$

This is the value at the beam tip (x = L), which is always to the right of the load application point.

Full piecewise deflection expression (derived in Appendix E):

$$\nu(x) = \frac{F_{eq}}{6EI} \begin{cases} -x^3 + 3\tilde{x}x^2 & x \leq \tilde{x} \\ 3\tilde{x}^2 x - \tilde{x}^3 & x > \tilde{x} \end{cases} \tag{42}$$

Derivation proceeds by:

  1. Finding M(x) from two cuts (left of and right of the point load)
  2. Integrating M(x)/EI twice to get ν(x)
  3. Applying fixed-end conditions: ν(0) = 0, ν'(0) = 0
  4. Applying continuity conditions at x = x̃: ν'_L(x̃) = ν'_R(x̃) and ν_L(x̃) = ν_R(x̃)

[STUDENT TASK — students derive deflection expressions for rectangular, triangular, and trapezoidal distributed loads in ASEN 3501.]

6.3 Geometric Load Approximation

The measured spanwise C_l distribution is compared to rectangular, triangular, and trapezoidal geometric shapes. Students apply engineering judgment to choose the best-fit shape for use in structural analysis and whiffle tree design.

Finite wing effects: Near the wing tips, C_l falls off due to tip vortex effects. The transition from the chord-wise infinite wing model (Milestone 2) to the finite wing (Milestone 3) is the physical reason the load distribution is not rectangular. Reference: Anderson §5.13.
